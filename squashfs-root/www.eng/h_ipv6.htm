IPv6 Help

Internet Protocol version 6(IPv6) is the next-generation internet layer protocol. This router implements the following transition mechanisms.

IPv4/IPv6 Dual Stack

In this mode, this router supports both IPv4 and IPv6 full function. Dual Stack can't work when in PPPoA or IPoA mode.

6to4 tunnel

6to4 uses encapsulation, meaning that it "wrap" IPv6 packets inside of an IPv4 shell to move them between networks over the current internet. In this mode, the WAN IP must be a public one. (RFC:3056)

NAT-PT(Network Address Translation - Protocol Translation)

NAT-PT performs stateful packet translation between internal IPv6 hosts and external IPV4 hosts.
