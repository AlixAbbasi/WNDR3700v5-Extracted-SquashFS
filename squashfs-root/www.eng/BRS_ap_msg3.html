Port forwarding

Port forwarding(sometimes called port mapping)allows access to your home network through the Intenet.If you have a device like a web server at home that others access remotely using a host name that you registered(usually through a dynamic DNS service) or using an IP address,you probably have the port forwarding function enabled.If you have never heard of this function,even if you have remote access to a device at home,it's likely you don't use this function
