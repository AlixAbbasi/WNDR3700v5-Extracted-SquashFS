ARP Protection/IP-MAC Binding
 
Enable IP-MAC Binding

IP-MAC binding protects your router from ARP attacks. With this function enabled, your router uses the LAN setup address reservation table under LAN setup as the static ARP. You can add, remove, or edit the IP-MAC binding pairs in the address reservation table. WARNING! Devices that are not registered in this table will NOT have internet access. Ensure that at least one LAN device's IP and MAC address pair is registered in this table. You can use the Edit button on this screen to directly access the address reservation table.
