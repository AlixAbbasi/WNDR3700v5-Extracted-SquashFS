Router Functions

The following router features are not available if you choose Access Point mode:

.Guest network
.Parental control
.Qos
.Access control
.Traffic meter
.Block sites
.Block services
.Static route

Advanced Functions

The following router functions are only available in the router mode,but they require configuration on your existing gateway:

.Port forwarding
.Port triggering
.IPv6
.UPnP/IGD
.Remote management
.VPN termination
